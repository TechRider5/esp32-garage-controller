## Next.js ESP32 Garage Controller

This project hosts a Next.js web UI for controlling an ESP32 garage controller with Firebase Auth and Realtime Database, including an admin panel to approve users.

### Environment Variables

Create a `.env.local` (not committed) with your Firebase and server secrets. See `.env.example` for keys. Configure the same variables in Vercel Project Settings â†’ Environment Variables.

### Develop

1. Fill in `.env.local`
2. `npm install`
3. `npm run dev`

### Deploy

Push to the repo connected to Vercel after setting env vars in Vercel. The API route `/api/iot-token` will be available to the ESP32.
This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

## Getting Started

First, run the development server:

